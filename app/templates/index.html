Food Allergy Detection

This AI model detects the presence of peanuts, tree nuts, dairy, eggs, and shellfish in unfamiliar foods. Useful when you're eating out and don't know what ingredients are in your food. Created by Flux-Blaze

 

                

                      The Process
                  

                    

                      

                        

                          Preparing Data
                        
For training data, we gathered images of common dishes that predominately feature peanuts, tree nuts, dairy, eggs, and shellfish by drawing bounding boxes around where those allergens appear in the image. Data augmentation methods like flipping or cropping the image are used to increase the dataset and prevent overfitting. We used RoboFlow for storing and annotating images as well as data augmentation.
                      

                      

                        

                          Training
                        

                        

                        First, we train the model on the training set to fit its parameters before using the validation set to tune the number of hidden layers. Finally, the model is evaluated using the test set. This process is repeated for hundreds of epochs in one round of training. After analyzing the metrics, we adjust the dataset and augmentations, going through more training rounds until we are satisfied with the model's performance. We used Yolov5 for the model's architecture and Google Colab to train it. 
                        

                      

                    

             

           

          

      

        
        
      

            

                

                  

                    Metrics
                  

                  

                    The graphs and charts below show the current model's performance in regards to its accuracy, precision, and recall.
                  
                  
                  

                    Confusion Matrix
                  

                  
                  

                    The Confusion Matrix shows how accurate the model was able to identify objects in our images. The diagonal line shows how accurate the model did when labeling true positives(correct labels), and as you can see our model did fairly well. The main problem was mostly the background False positives and negatives.
                  

                  

                    metrics/mAP
                  

                  

                  
                  
                  
                    

                  

                    
                  
                    

                  

                    This graph shows how mean Average Precision is affected as the model trains through more epochs. Higher precision means more correct positive identifications.
